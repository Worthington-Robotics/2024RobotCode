Detailed Description

A 1-1 2D lookup table for doubles that linearly interpolates between values not in the table

Mostly from https://github.com/FRC-5013-Park-Hill-Robotics/5013-RapidReact/blob/main/src/main/java/frc/robot/trobot5013lib/LinearInterpolator.java
